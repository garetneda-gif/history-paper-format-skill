# 《历史研究》引注规范标准 (Citation Standards)

## 1. 基本规则
- **注释形式**：页下脚注。
- **编号方式**：使用圈号 ①②③... 标识。
- **排序规则**：每页重新编号。
- **标注位置**：正文中注释序号置于标点符号之后。

---

## 2. 中文文献引注格式

### 2.1 著作 (Monographs)
- **标注顺序**：责任者与责任方式 / 文献题名 / 出版地点 / 出版者 / 出版时间 / 页码。
- **说明**：
  - 责任方式为“著”时可省略，其他（如编、校、整理）不可省略。
  - 译著：译者作为第二责任者置于题名后。
- **示例**：
  - 赵景深：《文坛忆旧》，上海：北新书局，1948年，第43页。
  - 实藤惠秀：《中国人留学日本史》，谭汝谦、林启彦译，香港：中文大学出版社，1982年，第11—12页。

### 2.2 析出文献 (Excerpted Works)
- **标注顺序**：责任者 / 析出文献题名 / 文集责任者与责任方式 / 文集题名 / 出版地点 / 出版者 / 出版时间 / 页码。
- **说明**：文集责任者与析出文献责任者相同时，可省去文集责任者。
- **示例**：
  - 鲁迅：《中国小说的历史的变迁》，《鲁迅全集》第9册，北京：人民文学出版社，1981年，第325页。

### 2.3 序言、前言、后记
- **标注顺序**：同析出文献或在著作后加“序言”等标识。
- **示例**：
  - 李鹏程：《当代文化哲学沉思》，北京：人民出版社，1994年，“序言”，第1页。

### 2.4 古籍 (Ancient Texts)
#### (1) 刻本
- **顺序**：责任者与责任方式 / 文献题名（卷次、篇名、部类）/ 版本 / 页码（a、b面）。
- **示例**：姚际恒：《古今伪书考》卷3，光绪三年苏州文学山房活字本，第9页a。

#### (2) 点校本、整理本
- **顺序**：责任者与责任方式 / 文献题名 / 卷次、篇名、部类 / 出版地点 / 出版者 / 出版时间 / 页码。
- **示例**：毛祥麟：《墨余录》，毕万忱点校，上海：上海古籍出版社，1985年，第35页。

#### (3) 影印本
- **顺序**：同上，注明“影印本”，页码可标明上、中、下栏。
- **示例**：杨钟羲：《雪桥诗话续集》卷5，沈阳：辽沈书社，1991年影印本，上册，第461页下栏。

#### (4) 析出文献
- **顺序**：责任者 / 析出文献题名 / 文集责任者 / 文集题名 / 卷次 / [丛书名] / 版本或出版信息 / 页码。
- **示例**：管志道：《续问辨牍》卷2《答屠仪部赤水丈书》，《四库全书存目丛书》，济南：齐鲁书社，1997年影印本，子部，第88册，第73页。

#### (5) 地方志
- **说明**：唐宋志可标作者；明清志冠以年号（如：万历《广东通志》）；民国志冠以“民国”。
- **示例**：万历《广东通志》卷15《郡县志二·广州府·城池》，《稀见中国地方志汇刊》，北京：中国书店，1992年影印本，第42册，第367页。

#### (6) 常用基本典籍
- **说明**：二十四史、《实录》、《全唐文》等可不标注作者。
- **示例**：《旧唐书》卷9《玄宗纪下》，北京：中华书局，1975年标点本，第233页。

#### (7) 编年体典籍
- **示例**：《清德宗实录》卷435，光绪二十四年十二月上，北京：中华书局，1987年影印本，第6册，第727页。

### 2.5 期刊 (Journal Articles)
- **标注顺序**：责任者 / 文献题名 / 期刊名 / 年期 (或卷期，出版年月) / 页码。
- **示例**：何龄修：《读顾诚〈南明史〉》，《中国史研究》1998年第3期。

### 2.6 报纸 (Newspaper Articles)
- **标注顺序**：责任者 / 篇名 / 报纸名称 / 出版年月日 / 版次。
- **示例**：李眉：《李劼人轶事》，《四川工人日报》1986年8月22日，第2版。

### 2.7 未刊文献 (Unpublished Materials)
#### (1) 学位论文、会议论文
- **顺序**：责任者 / 文献标题 / 论文性质 / 地点或学校 / 形成时间 / 页码。
- **示例**：方明东：《罗隆基政治思想研究（1913—1949）》，博士学位论文，北京师范大学历史系，2000年，第67页。

#### (2) 手稿、档案
- **顺序**：文献标题 / 形成时间 / 卷宗号或其他编号 / 藏所。
- **示例**：《傅良佐致国务院电》，1917年9月15日，北洋档案1011—5961，中国第二历史档案馆藏。

### 2.8 日记、书信 (Diaries & Letters)
- **顺序**：同著作，增加具体日期。
- **示例**：《翁同龢日记》第6册，光绪二十三年十月二十三日，陈义杰整理，北京：中华书局，1998年，第3058页。

### 2.9 转引 (Secondary Citations)
- **顺序**：责任者 / 原文献题名 / 原版本信息 / 原页码 / 转引自 / 转引文献责任者 / 转引文献题名 / 版本信息 / 页码。
- **示例**：章太炎：《在长沙晨光学校演说》，1925年10月，转引自汤志钧：《章太炎年谱长编》下册，北京：中华书局，1979年，第823页。

## 3. 外文文献引注格式

### 3.1 英文文献 (English)
- **专著**：Peter Brooks, *Troubling Confessions: Speaking Guilt in Law and Literature*, Chicago: University of Chicago Press, 2000, p.48.
- **析出文献 (期刊)**：Heath B. Chamberlain, "On the Search for Civil Society in China," *Modern China*, Vol. 19, No. 2, 1993, pp. 199-215.
- **档案**：Nixon to Kissinger, February 1, 1969, Box 1032, NSC Files, NPMP, National Archives II, College Park, MD.

### 3.2 日文文献 (Japanese)
- **专著**：小野五郎：『現代日本の産業政策一段階別決定刃のメカニズムー』、東京：日本経済新聞社、1999年、第27頁。
- **期刊**：梅津彰人：「キャンパスの文章---卒業論文」、『国文学 解釈と教材の研究』第40卷第2号、1955年、第71—74頁。

## 4. 特殊标注
- **再次引证**：简化为“责任者：题名，页码”。
- **间接引文**：以“参见”或“详见”引导。
- **夹注**：常用经典古籍（如《庄子》）可使用夹注，使用**楷体**。
- **电子文献**：责任者 / 题名 / 更新日期 / URL / 访问日期。
